# Student management system

## Project Overview

Student data management is essential in educational institutions. It helps organize records, reduce manual work, and improve administrative efficiency.

This project is a desktop-based Student Registration Management System built using JavaFX. It provides a structured interface to manage student records, subjects, grades, and employees with a modern UI and database integration.

## Features

User-friendly interface built with JavaFX  
Multi-page navigation for Student, Subject, Grade, and Employee sections  
Dynamic form handling with instant data updates  
Real-time data transfer between UI panels  
Student registration form with structured inputs  
Clean layout designed using Scene Builder  
Styled interface using CSS  
Data persistence using MySQL database  

## Core Modules

Student Module
Add and manage student records with structured form input.

Subject Module
Manage subject details and assignments.

Grade Module
Handle grading information and student performance records.

Employee Module
Maintain staff and employee data.

## Technologies Used

JavaFX  
Scene Builder  
NetBeans IDE  
MySQL Database  
CSS for UI styling  
JDBC for database connectivity  

## System Design

The application uses a multi-pane layout:  

Left Navigation Panel  
Contains menu options for switching between modules.  

Center Panel  
Displays active module forms and data entry screens.  

Right Panel
Shows live data updates after user actions.

## Database Integration

MySQL is used for data storage and management.  
All student and system records are stored securely.  
JDBC handles communication between Java application and database.
